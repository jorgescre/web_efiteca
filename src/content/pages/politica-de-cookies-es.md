Exploraremos cómo las cookies afectan nuestra privacidad y la funcionalidad en este sitio web.

Este servicio web utiliza cookies propias para el mantenimiento de la sesión del usuario que accede al servicio (cookies técnicas), personalizar las opciones que el usuario haya elegido (cookies de personalización) y de terceros de Google para mostrar el mapa y de Vimeo para mostrar videos.

Al usuario se le informa de la existencia de cookies propias y de terceros al entrar en el servicio y que se utilizan para el correcto funcionamiento del servicio web y mejorar la experiencia de usuario. El usuario deberá aceptar su uso en cuyo caso el mensaje no se mostrará más o cambiar la configuración de las cookies. Mientras no haga ninguna de las opciones, el mensaje se le seguirá mostrando.

**¿Qué es una Cookie?**
Una cookie es un pequeño fichero de texto que se almacena en el equipo del usuario a petición de la página web y se utiliza para que la página sea capaz de recordar acciones o preferencias del usuario a lo largo del tiempo.

**Clasificación de las Cookies en Efiteca**

| Tipo | Origen | Finalidad |
| :--- | :--- | :--- |
| **Cookies Técnicas** | Propias | Necesarias para la navegación y el mantenimiento de la sesión. |
| **Cookies de Personalización** | Propias | Permiten recordar las preferencias del usuario (idioma, configuración). |
| **Cookies de Terceros (Google)** | Terceros | Utilizadas para la integración de Google Maps en la web. |
| **Cookies de Terceros (Vimeo)** | Terceros | Utilizadas para la reproducción de vídeos integrados. |

**Gestión de Cookies**
Usted puede restringir, bloquear o borrar las cookies de este sitio web utilizando su navegador. Cada navegador tiene una configuración diferente:
*   Internet Explorer: [Configuración](https://support.microsoft.com/es-es/help/17442/windows-internet-explorer-delete-manage-cookies#ie=ie-10)
*   FireFox: [Configuración](https://support.mozilla.org/es/kb/habilitar-y-deshabilitar-cookies-sitios-web-rastrear-preferencias)
*   Chrome: [Configuración](https://support.google.com/chrome/answer/95647?hl=es)
*   Safari: [Configuración](https://support.apple.com/kb/ph21411?viewlocale=es_es&locale=es_es)

Tenga en cuenta que si desactiva las cookies, es posible que el sitio web no funcione correctamente.
